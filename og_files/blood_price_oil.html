// Three-tier model — each tier is a different accounting boundary
TIER_1 = direct_war_deaths / proven_reserves_contested
         = 300,000 / 145,000,000,000 = 2.07×10⁻⁶ deaths/bbl

TIER_2 = extraction_labor_deaths / production_volume (US, 2003–2013)
         = 1,189 / 47,000,000,000 = 2.53×10⁻⁸ deaths/bbl

TIER_3 = indirect_deaths_attributed / cumulative_production (era)
         = 100,000,000 / 1,500,000,000,000 = 6.67×10⁻⁵ deaths/bbl
